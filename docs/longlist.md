# AIコーディングCLIツール ロングリスト評価レポート

本レポートでは、エンタープライズ環境での利用を想定し、主要なAIコーディング向けCLIツール5製品（Claude Code、GitHub Copilot CLI、Cursor CLI、Gemini CLI、Codex CLI）についての比較・評価をまとめます。評価軸は「エンタープライズセキュリティ（データプライバシー・学習利用の有無）」「価格体系」「主要なCLI機能」の3点です。

## 1. Claude Code (Anthropic)
Anthropicが提供するターミナル上で自律的に動作するエージェント型CLIツールです。

*   **エンタープライズセキュリティ・データプライバシー:**
    *   **学習への利用制限:** 商用プラン（Team, Enterprise, API）では、送信されたコードやプロンプトがモデルの学習に使用されることはありません。
    *   **データ保持:** 基本は30日間保持ですが、Enterpriseプランではログを一切残さない「Zero Data Retention (ZDR)」モードの契約が可能です。
    *   **管理機能・コンプライアンス:** SOC 2 Type II認証取得済み。SAML/OIDCベースのSSOおよびSCIMによるユーザープロビジョニングの自動化、詳細なロールベースのアクセス制御（RBAC）、監査ログ（Audit Logs）、コンプライアンスAPIに対応しています。Amazon BedrockやVertex AI経由での利用もサポートされており、既存クラウドのセキュリティ境界内で運用できます。
*   **価格体系:**
    *   **Pro:** $17/月（標準的な利用枠）
    *   **Max 5x:** $100/月（Proの約5倍）
    *   **Max 20x:** $200/月（Proの約20倍）
    *   **Team Standard seat:** $20/ユーザー/月（年額契約）または $25/ユーザー/月（月次契約）。最小5ユーザー。Claude Codeを含みます。
    *   **Team Premium seat:** $100/ユーザー/月（年額契約）または $125/ユーザー/月（月次契約）。Standard seatの5倍の利用枠があり、重い利用向けです。
    *   **API (Pay-as-you-go):** Anthropic APIの標準従量課金で利用可能。加えて、Amazon BedrockまたはGoogle Vertex AIをバックエンドにして、それぞれのクラウド側の従量課金で運用することも可能です。
    *   **Enterprise:** カスタム（Team/Enterprise契約やAPI利用、クラウド経由利用を含めた個別設計）
*   **利用者目線での感想:**
    *   今は使っていないので情報が古いかもしれないですが、当時は設定や機能が複雑であったり玄人向けのイメージです。
    *   AGENTS.md問題などあまり標準化されない我が道をいく企業のイメージで、gemini cli、codex cliと同様な設定構成にするのが手間だった記憶があります。
    *   機械的なChatGPTの回答より、Claudeのフレンドリーな回答は好まれています。
*   **その他注目機能:**
    *   **Claude Code Security:** エージェントの自律的な動作に対する強力なガードレール機能。システム操作やシェルコマンドの実行前に、エンタープライズポリシーに基づいたセキュリティスキャンや明示的な承認（アプルーバル）フローを強制することが可能です。
    *   **Connectors (データソース連携):** Google Workspace、Slack、Confluence、Jiraなどの社内データソースと接続し、組織固有のナレッジを横断的に活用するエンタープライズ機能が提供されています。
    *   **Claude Code Desktop:** CLI操作に加えて、デスクトップアプリケーション環境からシームレスにエージェントを管理・監視できるUIツール。ローカル環境での高度な自律操作を視覚的にコントロールできます。
    *   **Claude Code Remote Control:** リモートサーバーやコンテナ環境（クラウドIDE等）上で稼働するClaude Codeエージェントを、ローカルの開発環境から安全に遠隔操作・オーケストレーションする機能。
    *   **Claude Code GitHub Actions:** CI/CDパイプラインに直接Claude Codeを組み込み、Pull Request作成時の自動コードレビューや自動修正、テスト生成などをトリガーベースで自律的に実行させる機能。
    *   **Claude Chrome Extension:** ブラウザ上で直接Claudeの機能を利用でき、社内WebシステムやSaaSツール上での作業をエージェントがシームレスに支援・自動化する拡張機能。
    *   **Claude CoWork (Enterprise機能):** チーム間でのプロンプトやコンテキストの共有、社内のナレッジベースとのシームレスな統合など、組織全体でのAI活用を促進する機能が展開されています。

## 2. GitHub Copilot CLI (GitHub)
「GitHub Copilot in the CLI」として統合された、ターミナル環境に特化したエージェントです。

*   **エンタープライズセキュリティ・データプライバシー:**
    *   **学習への利用制限:** 法人向けプラン（BusinessおよびEnterprise）では、ユーザーのコードやプロンプトがモデルの学習に使用されることはありません。
    *   **知的財産補償:** 著作権に関する法的リスクに対する補償（IP Indemnity）が含まれます。
    *   **管理機能:** 組織ポリシーによる一括管理（特定のコードベースの除外など）、詳細な監査ログ（Audit Logs）、DPA（データ保護合意）への準拠を提供します。
*   **価格体系:**
    *   **Copilot Pro:** $10/月 または $100/年（個人向け。Copilot CLI を含む）
    *   **Copilot Pro+:** $39/月 または $390/年（個人向け上位プラン。Copilot CLI を含み、Premium Requests が Pro の5倍）
    *   **Copilot Business:** $19/ユーザー/月（組織管理、ポリシー制御、IP補償含む）
    *   **Copilot Enterprise:** $39/ユーザー/月（Businessの機能に加え、組織独自のナレッジベース活用やカスタムモデル調整が可能）
*   **利用者目線での感想:**
    *   gpt, claudeなど様々なモデルが利用できます。
    *   利用料が一般的なコンテキスト量ではなく、エージェントとの一回のやり取りの回数で利用計算されやり方によってはコストパフォーマンスが非常に良くなるようです。
*   **その他注目機能:**
    *   **Copilot Workspace / ナレッジ検索:** IssueからPull Requestまでの工程全体をAIと人間が協調して進める統合ワークスペース環境や、エンタープライズリポジトリ全体のインデックス化による高度なコードベース検索が可能です。
    *   **GitHub Agent HQ (エージェント・プラットフォーム):** GitHubをAIエージェントの「司令塔（本部）」として位置づける構想。Copilotだけでなく、Claude Codeやサードパーティ製、あるいは自作のAIエージェントをGitHubのIssueやPRに統合し、それらを一括して実行・監視・オーケストレーションできる環境を提供します。企業向けの「Enterprise Agent Control Plane」により、エージェントの権限管理やセキュリティポリシーの適用、実行ログの監査をプラットフォーム側で一元的に行うことが可能です。

## 3. Cursor CLI (Cursor)
IDEとして有名なCursorが提供する、強力なターミナル用エージェント機能（Cursor CLI Agent）です。

*   **エンタープライズセキュリティ・データプライバシー:**
    *   **学習への利用制限:** 「Privacy Mode」を有効にすることで、コードがサーバーに永続保存されず学習にも使用されません（Business/Enterpriseプランではデフォルトで強制適用）。OpenAIやAnthropicなどのプロバイダーともZero Data Retention契約を締結しています。
    *   **管理・コンプライアンス:** SOC 2 Type II 認証済み。SAML/SSO対応、SCIMによるプロビジョニング、詳細な監査ログのストリーミング転送（Datadog等）に対応しています。
*   **価格体系:**
    *   **Pro:** $20/月（個人向け。エージェント利用上限の拡張、最先端モデル、MCP、スキル、フック、クラウドエージェントを含む）
    *   **Pro+:** $60/月（個人向け上位プラン。Pro の全機能に加え、OpenAI / Claude / Gemini モデルで使用量 3 倍）
    *   **Ultra:** $200/月（個人向け最上位プラン。Pro の全機能に加え、OpenAI / Claude / Gemini モデルで使用量 20 倍、新機能への優先アクセス）
    *   **Teams:** $40/ユーザー/月（チーム向け。共有、利用分析、組織全体の Privacy Mode 制御、RBAC、SAML/OIDC SSO）
    *   **Enterprise:** カスタム（Teams の全機能に加え、使用量プール、請求書/PO 請求、SCIM、AI コード追跡 API、監査ログ、詳細な管理者権限）
*   **利用者目線での感想:**
    *   利用経験がなくなんとも言えません（インターネットでも情報を全く聞かないので利用者は少ないと思われます）
    *   GitHub Copilotと同じく様々なモデルが利用できます。
*   **その他注目機能:**
    *   **高速インデックスとカスタムDocs:** プロジェクト全体の高速なインデックス作成により、大規模なコードベースでも瞬時に関連ファイルを見つけ出します。また、社内ドキュメントや最新ライブラリの公式ドキュメント（Docs）をカスタム追加してコンテキストに含める機能が強力です。

## 4. Gemini CLI / Gemini Code Assist (Google)
Google Cloudの強固な基盤上で動作するAIコーディングツールおよび自律型CLIです。

*   **エンタープライズセキュリティ・データプライバシー:**
    *   **学習への利用制限:** エンタープライズ利用において、入力したコードやプロンプトがモデルの学習に使用されることはありません。
    *   **エンタープライズ管理:** VPC Service ControlsやCMEK（顧客管理暗号鍵）などの高度なセキュリティ設定に対応。Google Cloudの堅牢なインフラ上で管理されます。
    *   **補償と安全性:** 生成コードに関する著作権侵害リスクへの補償が適用。DockerやmacOS Seatbeltを利用したサンドボックス実行が可能です。
*   **価格体系:**
    *   **Google AI Plus（個人向け、一部地域）:** 公式サブスクリプションページに掲載あり。地域によって提供状況と価格が異なり、例えば一部ページでは月額 $9.99 相当の中間プランとして案内されています。Geminiアプリ、追加のAIクレジット、Googleアプリ連携、ストレージ拡張などが含まれます。
    *   **Google AI Pro（個人向け）:** $19.99/月。1か月無料トライアルあり。公式のサブスクリプションページでは、Gemini CLI / Gemini Code Assist IDE拡張の「より高い日次リクエスト上限」が含まれると案内されています。
    *   **Google AI Ultra（個人向け）:** $249.99/月。米国向け公式ページでは最初の3か月は $124.99/月。Gemini CLI / Gemini Code Assist IDE拡張の「最高の日次リクエスト上限」が含まれると案内されています。
    *   **Standard Edition:** $22.80/ユーザー/月（月次契約）または $19/ユーザー/月（12か月契約）。Gemini CLIの利用を含み、1日1,500リクエスト上限。
    *   **Enterprise Edition:** $54/ユーザー/月（月次契約）または $45/ユーザー/月（12か月契約）。Gemini CLIの利用を含み、1日2,000リクエスト上限。初回導入時は最低10ライセンス。
    *   **Pay-As-You-Go:** Vertex AIやGemini APIキーを利用した従量課金制
*   **利用者目線での感想:**
    *   コーディングにおいてのモデル性能がgpt, claudeと比べると低いです。
    *   Google AI Proプラン利用者ですが、利用制限が1日間隔となっているにも関わらず、同価格帯のChatGPT Plusと比べるとすぐに利用制限がかかり非常にコストパフォーマンスが悪い印象です。
*   **その他注目機能:**
    *   **Google Jules / エコシステム統合:** Googleが社内向けに開発・運用しているAIエージェント「Jules」で培われた高度なGitHub/GitLab連携やワークフロー自動化の知見が活かされており、Google Cloud（Vertex AI等）のエンタープライズ機能と強力に統合されています。

## 5. Codex CLI (OpenAI)
次世代のコーディングエージェントとして展開が進む、OpenAIのターミナル向け自律実行ツールです。

*   **エンタープライズセキュリティ・データプライバシー:**
    *   **学習への利用制限:** ChatGPT EnterpriseおよびBusinessプランでは、入力データや出力コードがOpenAIのモデル学習に使用されることはありません。
    *   **管理機能:** SAML SSO、SCIM対応。OpenAI Admin CLIを用いたユーザー・プロジェクト・APIキーの一元管理や、詳細な監査ログ、コストガバナンス（予算設定）が可能です。SOC 2 Type II準拠。
*   **価格体系:**
    *   **ChatGPT Go:** $8/月（個人向け低価格プラン。Codexは期間限定で利用可能）
    *   **ChatGPT Plus:** $20/月（個人向け。Codexを利用可能だが、組織管理やSSOは対象外）
    *   **ChatGPT Pro:** $200/月（個人向け。Plusより高い利用上限でCodexを利用可能）
    *   **ChatGPT Business:** $25/ユーザー/月（年額契約）または $30/ユーザー/月（月次契約）。中小規模チーム向け、データ学習なし
    *   **ChatGPT Enterprise:** 個別見積り（大規模組織向け、高速アクセス、SLA提供）
    *   **API (Pay-as-you-go):** トークン数に応じた従量課金（GPT-4o等の最新推論モデルを利用）
*   **利用者目線での感想:**
    *   GPT‑5.4というトップレベル性能に加え、Plusプランでも利用制限にかかりづらくコストパフォーマンスが非常に良いです。
    *   他のcli型コーディングエージェントはnodejs製ですが、Codex CLIはRust製で動作が非常に軽快です。
*   **その他注目機能:**
    *   **OpenAI Frontier:** 企業内で多数の自律型AIエージェントを構築・管理・運用するためのエンタープライズ向けプラットフォーム。既存の業務システムや独自データベースと連携し、エージェントの権限管理、実行環境（サンドボックス）、パフォーマンス評価までを一元管理します。
    *   **OpenAI Operator:** エージェントがユーザーに代わってブラウザやデスクトップ環境を直接操作（コンピューター・ユース）し、コーディングから動作確認、外部ツールのGUI操作までをシームレスに自動化する機能。
    *   **Codex Web / エンタープライズデータ統合:** OpenAIのエンタープライズプラットフォームを活用し、GitHub、Slack、Google Driveなどの社内データをコネクタ経由でエージェントに読み込ませ、社内の固有知識に基づいた高度な自律実行をサポートします。
    *   **ChatGPT Atlas:** 企業内のあらゆるデータソース（コード、ドキュメント、チャット履歴等）を統合し、エージェントが参照可能な巨大なナレッジグラフを構築する基盤機能。
    *   **Codex Security:** エージェントが生成したコードや実行するコマンドに対する、エンタープライズ水準の自動脆弱性スキャンとポリシー適用機能。
    *   **Codex App:** デスクトップやモバイル環境から、プロジェクトの状況確認やエージェントへの指示・承認（アプルーバル）を行えるコンパニオンアプリ。

## 6. Devin (Cognition AI)
Cognition AIが提供する、自律型のAIソフトウェアエンジニアです。CLI専用というよりは独自のワークスペース（UI/CLI複合）で動作しますが、自律型エージェントの代表格として比較に含めます。

*   **エンタープライズセキュリティ・データプライバシー:**
    *   **SOC 2 Type II 認定**: 取得済みであり、厳格なセキュリティおよびプライバシー基準に準拠しています。
    *   **学習への利用制限**: エンタープライズプラン等において、顧客のデータやコードがモデルのトレーニングに使用されることはありません。
    *   **エンタープライズ管理**: 顧客の仮想プライベートクラウド（VPC）内へのデプロイが可能であり、SSOや詳細な監査ログなど、エンタープライズ水準の管理機能を提供します。
*   **価格体系:**
    *   **Core:** 従量課金制。$20から開始でき、月額コミット不要。ACU（Agent Compute Unit）単価は $2.25/ACU。
    *   **Team:** $500/月。250 ACU が含まれ、含有分の実質単価は $2.00/ACU。必要に応じて追加ACUをオンデマンド消費可能です。
    *   **Enterprise:** 個別見積り。VPC デプロイ、SAML/OIDC SSO、集中管理などのエンタープライズ機能を含みます。
*   **主要なCLI/エージェント機能:**
    *   コードの提案にとどまらず、自然言語による指示から「計画・実装・テスト・デバッグ・デプロイ」までのプロセス全体を自律的に遂行します。
    *   独自のサンドボックス環境内で、シェル（ターミナル）、コードエディタ、ブラウザを自由に使用し、エラーが起きればログを見て自らデバッグを行います。
*   **利用者目線での感想:**
    *   利用経験がなくなんとも言えません。
*   **その他注目機能:**
    *   **Ask Devin / DeepWiki:** 社内のコードベースやドキュメントに関する質問に即座に答える「Ask Devin」機能や、プロジェクトのコンテキストを自動的に学習・蓄積する「DeepWiki」といったナレッジ管理機能が提供されています。

---

**総括:**
エンタープライズ環境での採用にあたっては、いずれのツールも法人向けプラン（Enterprise/Team等）を選択することで「データ学習への非利用」や「SSO/監査ログ」などの基本要件を満たします。選択のポイントは、既存のインフラ環境（AWS/GCP/Azure）、エージェントの自律性の高さ（CursorやClaude Codeなど）、そしてコストモデル（固定か従量課金か）のフィット感となります。
